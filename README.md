
# Trader Behavior & Market Sentiment Analysis

This project explores the relationship between trader performance and Bitcoin market sentiment (Fear–Greed Index).

## Folder Structure
- **notebook_1.ipynb** – Complete analysis (Steps 1–4).
- **csv_files/** – Contains input datasets (trader and sentiment CSVs).
- **outputs/** – Folder for optional visual outputs (plots, charts).
- **ds_report.pdf** – Final summarized report of insights.
- **README.md** – Project overview and submission notes.

## Summary of Findings
- Most trades occurred during Fear and Greed phases.
- Traders took larger positions in Fear but were more active during Greed.
- Sentiment correlated weakly (r ≈ –0.1) with PnL, suggesting better discipline during cautious markets.
- Fees correlated strongly (r ≈ 0.98) with trade size, as expected.
- Market sentiment influenced trader behavior more than direct profitability.
